bool send_request_with_ack ( int sockfd,
const sockaddr_in & serverAddr,
const Packet & packet )

Sends a request packet to the server and waits for acknowledgment.

This function will retry sending the request up to 3 times if no acknowledgment is received. The user will be prompted to retry if the maximum number of attempts is reached. The function returns true if the acknowledgment was received; false otherwise.

Parameters
sockfdThe socket file descriptor.
serverAddrThe server address structure.
packetThe packet to send.
Returns
True if the acknowledgment was received; false otherwise.
